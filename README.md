# phpqrcode

PHP QRCode integration for Logiks


//Use it as follows:

loadVendor("phpQRCode");

QRcode::png('code data text', 'filename.png'); // creates file
QRcode::png('some othertext 1234'); // creates code image and outputs it directly into browser
